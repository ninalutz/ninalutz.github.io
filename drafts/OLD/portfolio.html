In Progress Works

Here are some things I've been doing recently...


SpaceWork

SpaceWork is a multitouch drawing desk that I am currently working on for my 6.835 (Intelligent MultiModal User Interfaces) class project.



MedWatch

MedWatch is an IOT watch that reminds users to take their medication. This is all controlled by a server where the user and doctor can see medication patterns and add medications. The watch includes an OLED display, haptic feedback, and a two step medication verificiation process that incldues RFID and gesture recognition. This is my final project for my Embedded Systems class at MIT.


Philips Urban Lighting

This is a model I'm working on in my lab. I'm implementing a force based pedestrian model to see the consequences of persuasive urban lighting in the city. This project is expected to be completed in Fall 2018 and goes with my other research ventures in the CityScope development team at the MIT Media Lab Changing Places Group.



GSK Smart Lab

This is a current research project I'm working on in my lab, Changing Places. This project seeks to develop a tool and model to simulate how scientists are collaborating together in a lab space and how we can optimize this collaboration. We seek not only to implement an accurate model but also a tangible interface in the form of a modified tactile matrix which will also incorporate archiecture drawing interventions.
