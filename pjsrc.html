PJ's RC Mobile App

I'm currently doing a lot of fun experimenting with computer vision, particularly with part identification. We know what a part looks like from its 3D model, so it's not hard to see when its broken. This is easy with one part, but a little harder when looking a variety of parts in a region on a radio control car.


Using the SIFT algorithm and some hand cooked algorithms and machine learning implementations I'm developing an iOS app that allows the user to take a photo of the region of the car that's giving them issues, identifies broken parts, and lets the user purchase a replacement.


This is a project I've been having a lot of fun with and I'm set to finish and pitch my application during IAP (MIT's January break) 2017.
