Singapore

Exploring different ways to transform geospatial data between different coordinate systems. Rasterizing geospatial data to a Lego coordinate system for fine grained pedestrian movement and integration with tangible user interface.

This was a one month project with Singaporean planners, under my supervisor Ira Winder. The goal was to design a model to access pedestrian accessibility in a region of Singapore according to age demographic. We did this with a tangible user interface where users could add different amenities and explore the accessibility according to age or amenity. This was done by math modeling accessibility in the region according to the paths that the pedestrian can take. You can read more about our lab's methodology in this write up.

My role in this project was looking at how we rasterize and transform geospatial data between different coordinate systems to get a graph where different pedestrians in our simulation could navigate. I also helped with the design workshops in this project for design the topologies of the amenity pieces for the tangible user interfaces and the overall interface and model. I really enjoyed working with the planners and learned so much about planning in a different part of the world.
