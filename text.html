L-Systems Textile

An L-System is an algorithm that involves an axiom and formal grammar for how a system grows and behaves. L-Systems often help describe and model biological growth like trees or a circulatory system, as well as complex fractals and systems.

This is an example of an output from a script of a simple L-System where each generation is grown at the same angle. As you can probably tell, I use Processing as my sketchbook for many things in my design process.

To make a textile I chose to write code that generated multiple L Systems and let the user layer them and export the canvas into a PDF.

This PDF can be modified in Rhino and saved as an STL.

This of course can be 3D printed. For this print I used FormLabs' Form 2 Printer with Flexible Resin. Here are some photos of a swatch of my L-Systems textile.

Detailed code and instructions will be coming to my GitHub soon; so stay tuned!
