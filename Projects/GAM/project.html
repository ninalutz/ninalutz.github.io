Gender Affirming Makeup (GAM)

A tool for empowerment and expression.

GAM

Present, Masters Thesis
  • Software
  • OpenCV
  • Simulation
  • Computational Geometry

What if we could leverage technology and modern cosmetics for empowerment?

The goal of this project is to use mathematics and computation to create a tool that will allow a user to add more feminine, masculine, or androgynous traits to themselves with cosmetics. After interacting with the tool, the user will receive a list of cosmetics and then follow along with the tool in real time while they apply their makeup.

This tool is part of my Masters Thesis at the MIT Media Lab. It lives here.
